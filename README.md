## Usage
    <build>
        <plugins>
            <plugin>
                <groupId>walkingdevs</groupId>
                <artifactId>rebelator</artifactId>
                <version>3</version>
            </plugin>
        </plugins>
    </build>

    <repositories>
        <repository>
            <id>bintray</id>
            <url>http://dl.bintray.com/walkingdevs/mvn</url>
        </repository>
    </repositories>

## Run
    mvn rebelator:do
